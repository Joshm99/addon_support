Installation & Updating

How to install, update, and troubleshoot JM Addons on Minecraft Bedrock Edition.

Installation

  1. Download the .mcaddon from CurseForge.
  2. Double-click the file to automatically import the addon into Minecraft (Close Minecraft first for the best result).
  3. Create or open a world and go to Behavior Packs in world settings.
  4. Activate the Behavior Pack. The resource pack is linked automatically.
★
Tip

These steps apply to all JM Addons. Each addon ships as a single .mcaddon file that bundles both the behavior pack and resource pack together.

Updating

To update an addon to a newer version:

  1. Download the latest .mcaddon from CurseForge.
  2. Close Minecraft completely.
  3. Double-click the new .mcaddon file to import it. Minecraft will replace the old version automatically.
  4. Open your world — the updated addon is already active.

Keeping your data

Updates preserve your world data. Scoreboards, tags, dynamic properties, and any in-game state created by the addon remain intact after updating. You do not need to deactivate or remove the old version first.

Troubleshooting

  • Addon not appearing in Behavior Packs? Make sure Minecraft was fully closed before importing. Try importing the .mcaddon file again.
  • Resource pack not loading? The resource pack is linked to the behavior pack and should activate automatically. If it doesn't, check that it's listed under Active Packs in your world's Resource Packs settings.
  • Features not working after update? Close and reopen the world. Some script-based features need a fresh world load to initialize.
