# JM Addons — Documentation Site

Source for [jmaddons.support](https://jmaddons.support/), a static documentation site for Minecraft Bedrock addons built by JM.

## Structure

```
index.html              — Home page (addon card grid)
installation.html       — Installation & updating guide
addons/
  admintools.html       — AdminTools docs
  notice-board.html     — Notice Board docs
  mail-delivery.html    — Mail Delivery docs
  tag-tool.html         — Tag Tool docs
style.css               — Main stylesheet
search.css              — Search UI styles
search.js               — Client-side search
search-index.json       — Search content index
sitemap.xml             — Sitemap
images/                 — Logos, icons, and hero images
```

## Addons

| Addon                                      | Version | Description                                                            |
| ------------------------------------------ | ------- | ---------------------------------------------------------------------- |
| [AdminTools](addons/admintools.html)       | v2.1.0  | Full admin toolkit — moderation, claims, ranks, and more               |
| [Notice Board](addons/notice-board.html)   | v1.0.4  | In-game community notice board with categories, search, and comments   |
| [Mail Delivery](addons/mail-delivery.html) | v1.0.0  | Postal system for sending items, packages, and letters between players |
| [Tag Tool](addons/tag-tool.html)           | v1.0.0  | Lightweight in-game player tagging utility                             |
